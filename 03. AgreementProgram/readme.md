# Agreement Program in C

**Project Name:** Agreement Program  
**Language:** C  
**Concepts:** `getchar()`, Logical OR (`||`), Character Input  

---

## Description

This program asks the user a simple yes/no question to check for agreement. It accepts four possible responses: `'y'`, `'Y'`, `'n'`, `'N'`. Using the logical OR operator (`||`), it checks if the input matches any valid response and prints whether the user agrees or disagrees.

---

## Features

- Handles both uppercase and lowercase inputs (`y/Y/n/N`).  
- Uses `getchar()` for single character input.  
- Demonstrates the use of logical OR (`||`) operator.  
- Beginner-friendly program for learning C basics.
