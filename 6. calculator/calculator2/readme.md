# Division Program

In this program, the user is asked for an 'x' and 'y' value, the the program divides it by showing the answer.

Using (float) and %f, the program is able to give a float answer if it has to.
This program also specifies the number of float it should render.

## Features
> cs50.h,
> get_int,
> (float),
> %f,
> printf
